# מחשבון ציוני ספורט - בית הספר הריאלי

אתר לחישוב ציוני מבחני ספורט לתלמידי תיכון.

## הרצת האתר

**לשימוש בפועל, יש להעלות את כל הקבצים לשרת אינטרנט.**

## קבצי תצורה

האתר ניתן להרחבה מלאה על ידי עריכת קבצי CSV:

### 1. `options.csv` - אפשרויות בטפסים ומידע על מבחנים

קובץ זה מגדיר את כל האפשרויות בתפריטים הנפתחים ומידע על המבחנים:

```csv
field,value,label,description,input_format
grade,9,כיתה ט',,
grade,10,כיתה י',,
gender,male,זכר,,
gender,female,נקבה,,
test_type,2000m,ריצת 2000 מטר,הזן את הזמן בפורמט דקות:שניות (לדוגמה: 8:30),time
test_type,pullups,מתח,הזן את מספר החזרות שביצעת,count
test_type,4x10m,ריצת 4x10 מטר,הזן את הזמן בפורמט שניות (לדוגמה: 12.5),seconds
```

**העמודות:**
- `field` - סוג השדה (grade/gender/test_type)
- `value` - הערך הפנימי
- `label` - התווית בעברית
- `description` - הוראות למשתמש (רק למבחנים)
- `input_format` - סוג הקלט (רק למבחנים)

**ערכי `input_format` אפשריים:**
- `time` - זמן בפורמט דקות:שניות, נמוך יותר = טוב יותר (לדוגמה: 8:30)
- `seconds` - זמן בשניות, נמוך יותר = טוב יותר (לדוגמה: 12.5)
- `count` - מספר חזרות (מספרים שלמים), גבוה יותר = טוב יותר (לדוגמה: 20)
- `decimal` - ערכים עשרוניים, גבוה יותר = טוב יותר (לדוגמה: 2.5) - למרחקים, משך זמן, רמות

### 2. `scores.csv` - נתוני ציונים

קובץ זה מכיל את הסטנדרטים לציון 100 עבור כל קומבינציה:

```csv
test_type,gender,grade,score_for_100
2000m,male,9,8:30
2000m,female,9,10:00
pullups,male,9,15
```

**העמודות:**
- `test_type` - סוג המבחן (חייב להתאים ל-`test_type` ב-`options.csv`)
- `gender` - מגדר (`male` או `female`)
- `grade` - שכבת גיל (9, 10, 11, 12)
- `score_for_100` - הביצוע הנדרש לציון 100

## הוספת סוג מבחן חדש

לדוגמה, נוסיף מבחן "קפיצה לגובה:

1. הוסף ל-`options.csv`:
```csv
test_type,high_jump,קפיצה גבוה,הזן את הגובה במטרים (לדוגמה: 1.5),seconds
```

2. הוסף ל-`scores.csv` (עבור כל קומבינציה של מגדר ושכבה):
```csv
high_jump,male,9,1.3
high_jump,male,10,1.4
high_jump,female,9,1.1
high_jump,female,10,1.2
...
```

זהו! המבחן החדש יופיע אוטומטית באתר.
